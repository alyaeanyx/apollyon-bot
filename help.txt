*Available commands*:
**{p}help:** display this message
**{p}list:** list all available feeds
**{p}add <feed>:** subscribe this channel to a specific feed
**{p}remove <feed>:** unsubscribe this channel from a specific feed
**{p}clear:** unsubscribe this channel from all feeds
**{p}mention <feed> (none|@everyone|@here|@<role/user>):** set which role should be mentioned in case of an update
